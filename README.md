# Cardinal

## Quick Start

1. Clone the repository:
```bash
  git clone https://github.com/0x0Dx/cardinal.git
  cd cardinal
```

2. Create your config:
```bash
  cp .env.example .env
  nano .env  # CHANGE THE PASSWORD!
```

3. Start Cardinal:
```bash
  docker-compose up -d --build
```

4. Access at: `http://localhost:8000`
